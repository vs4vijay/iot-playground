Real-time Weather Display with OpenWeatherMap API

About this firmware

Weather Micro Station is a modular, high-performance weather display system for the LILYGO T-Display S3, featuring real-time weather data from OpenWeatherMap API with smooth animations and professional UI.

Features:

  • Real-time weather data fetched every 3 minutes
  • 18 different weather condition icons
  • 40 FPS scrolling ticker with smooth transitions
  • Display brightness control with hardware buttons
  • Automatic NTP time synchronization

Device: LilyGo T-Display S3 (ESP32-S3)
Version: 6f63e83

Flash Firmware to Device

üìã Instructions

  1. Get a free API key from OpenWeatherMap.org
  2. Connect your LilyGo T-Display S3 to your computer via USB cable
  3. Click the "CONNECT" button above
  4. Select your device from the popup (usually shows as "USB JTAG/serial debug unit")
  5. Click "INSTALL" and wait for the flashing process to complete
  6. Configure your WiFi and API credentials (see configuration section below)
  7. Device will reboot and start displaying weather data

‚öôÔ∏è Post-Installation Configuration

After flashing the firmware, you need to configure your settings. There are two options:

  1. Option 1 - Recompile with PlatformIO:
    • Clone the repository: git clone https://github.com/sfrechette/weather-micro-station
    • Copy include/secrets_template.h to include/secrets.h
    • Edit include/secrets.h with your WiFi and API credentials
    • Build and upload: pio run --target upload
  2. Option 2 - Serial Configuration: Use the serial monitor at 115200 baud to configure settings (if supported by firmware)
